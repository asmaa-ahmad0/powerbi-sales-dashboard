# Sales Dashboard | Power BI

## Overview

This project is an interactive Sales Dashboard built using Microsoft Power BI.

It provides insights into:

- Sales Performance
- Profit Analysis
- Customer Segmentation
- Product Performance
- Regional Analysis

---

## Dashboard Features

### Page 1
- KPI Cards
- Sales by Category
- Sales by Ship Mode
- Customer Segment Analysis
- Profit by Region
- Quantity by Region
- Monthly Sales Trend

### Page 2
- Best Customer
- Best Product
- Customer Matrix
- Decomposition Tree
- Top 10 Products

---

- Microsoft Power BI
- DAX
- Power Query
- Data Modeling
